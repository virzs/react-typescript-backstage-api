## NestJs + typeORM接口项目

## 安装模块

```bash
$ npm install
//或
$ yarn
```

## 启动项目

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 测试

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
